Tao Analysis Flashcards

Welcome! This website provides spaced repetition flashcards for Terence Tao's Analysis I, via Orbit. Flashcards look like this:

Orbit was intended to be used for online writing so that the reader can do the flashcards as they are reading the text. Since Analysis I was not written as an online text with embedded flashcards, the flashcards must be hosted externally on this site. That means that you, the reader, will need to do a little bit of extra work to remember to come back to this site after reading each section of the book. The outline below is to help you know when to do which set of flashcards. After you do a set of flashcards, reminder emails will be sent to you so you will not need to keep visiting this site. For more information, please see the Orbit website.
